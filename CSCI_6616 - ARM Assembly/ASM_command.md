## ARM Commands

| Command | Description | 
| ------- | ----------- |
| MOV | Moves within registers |
| CMP | Compares r0, #16, takes R0-16 = 0? and updates status register |
| ---- | ---------------|
| Branching |           |
| beq | branch on equal of previous comparison operation, relys on status reg flag | 
| ble | Branch on less than |
